Privacy Policy — Sovara

Last updated: April 2, 2026

Sovara works 100% offline. No internet connection is ever used. No account required. No server. No cloud. Your data never leaves your device — ever.

No internet No account No server No ads No tracking Works in airplane mode

1. Overview

Sovara is a local Android application that monitors access to your device's microphone and camera by other applications. Unlike most security apps, Sovara requires no internet connection and sends absolutely nothing outside your device.

There is no backend server, no cloud storage, no analytics, no crash reporting, and no advertising network. Sovara is a tool that runs entirely on your phone — nothing more.

2. What data is stored

Sovara stores the following data exclusively on your device, in local storage that only Sovara can access:

  • Timestamps of microphone and camera access events
  • Name and package identifier of the application that triggered the event (only if you grant the optional Usage Access permission)
  • Duration of each access event
  • Your in-app settings and preferences

This data is never transmitted anywhere. It exists only on your device and is fully under your control.

3. What data is NOT collected

  • No audio is recorded — ever
  • No photos or video are captured — ever
  • No location data
  • No device identifiers
  • No usage statistics sent to any server
  • No personal information of any kind

4. Permissions used

RECORD_AUDIO

Used only to detect when another application accesses the microphone. Sovara never records, processes, or stores any audio content.

PACKAGE_USAGE_STATS

Optional. Used to identify which application was in the foreground when a microphone or camera access occurred. Must be granted manually by the user in Android system settings. Sovara works without it — app names may simply appear as unknown.

FOREGROUND_SERVICE / FOREGROUND_SERVICE_MICROPHONE

Required to run the monitoring service continuously in the background. Android requires a visible notification when a foreground service is active — this is the "Protection active" notification you see.

POST_NOTIFICATIONS

Used to display real-time alerts when microphone or camera access is detected by another application.

RECEIVE_BOOT_COMPLETED

Used to restart the monitoring service automatically after the device reboots, if you had previously enabled protection.

VIBRATE

Used to vibrate the device as part of alert notifications, if you have enabled vibration alerts.

5. No third-party services

Sovara contains zero third-party SDKs that transmit data externally. No Google Analytics, no Firebase, no Crashlytics, no Facebook SDK, no advertising network of any kind. The application is built exclusively with Android standard libraries and Jetpack components — all of which run locally.

6. Data deletion

You are in full control of your data at all times:

  • Delete history: History screen → Delete button — erases all recorded events instantly
  • Uninstall: Removes all data permanently from your device

There is no account to delete, no server-side data to request removal of — because nothing was ever stored anywhere but your device.

7. Children

Sovara does not collect any personal information from any user, including children. As no data leaves the device, this application is safe for users of all ages.

8. Changes to this policy

If this privacy policy is updated, the new version will be published at this same URL with an updated date. No changes will affect data that was never collected in the first place.

9. Contact

For any questions about this privacy policy, contact the developer via the Google Play Store listing page.
